说明¶

本文使用 Sphinx Doc 生成

Mac 安装¶

# Sphinx
$ brew install python3
$ pip3 install -U Sphinx

# markdown
$ pip3 install --upgrade recommonmark

# theme
$ pip3 install sphinx_rtd_theme
$ pip3 install msmb_theme

使用快速工具来初始化项目¶

$ sphinx-quickstart

欢迎使用 Sphinx 2.3.1 快速配置工具。

和“build”两个独立的目录。
